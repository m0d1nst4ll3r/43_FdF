# FdF

Rendering a 3D mesh using mlx.

## Instructions

1. Clone repo
```bash
git clone https://github.com/m0d1nst4ll3r/43_FdF.git
cd 43_FdF
```
2. Clone mlx
```bash
git clone https://github.com/42paris/minilibx-linux.git
```
3. Clone libft
```bash
git clone https://github.com/m0d1nst4ll3r/43_mylibft.git
```
4. make `make`
```bash
make
```
5. Run fdf
```bash
./fdf
```
